> **"코딩은 AI가 합니다. 당신의 아이는 '무엇을 할지' 결정만 하면 됩니다."**

## 섹션 1: 프로그램 계획 (Plan)
### 1.1 프로그램 비전 및 철학
**"코딩 기술자가 아닌, AI 시대를 지휘할 설계자(Architect)를 키웁니다."**
이 프로그램은 프로그래밍 문법을 가르치는 교육이 아닙니다. AI라는 압도적인 지능을 가진 도구를 **자신의 손발처럼 부릴 수 있는 '장악력'** 을 기르는 리더십 과정입니다.
| 기존 교육 | 본 프로그램 |
|-----------|------------|
| 코드를 작성하는 **노동자(Laborer)** | 설계도면을 그리는 **건축가(Architect)** |
| 정답을 찾는 학습 | **문제를 정의하는** 훈련 |
| 개인 스킬 습득 | 팀(부모+자녀) 협업 경험 |
| AI를 두려워하는 세대 | **AI 위에서 노는 세대** |
핵심 역량 3가지:
1. **맥락 설계(Context Engineering)** — AI에게 올바른 지시를 내리기 위한 논리적 사고
2. **의사결정(Decision Making)** — 무엇을 만들고, 무엇을 포기할지 판단하는 능력
3. **협업 리더십(Collaborative Leadership)** — 사람(부모)과 AI를 동시에 지휘하는 경험
---
### 1.2 핵심 컨셉: 부모-자녀 현업 강사 모델
```
┌─────────────────────────────────────────────────┐
│       하나의 팀 = 하나의 스타트업       │
│                         │
│  자녀 (상상력 담당)   부모 (정리 담당)  │
│  ┌─────────────────┐  ┌──────────────────┐  │
│  │ 무한한 상상력   │  │ 현실 감각     │  │
│  │ "하늘 나는 고양이" │  │ "날개는 어떤 버튼?" │  │
│  │ 아이디어 발산   │  │ 구조화·구체화   │  │
│  │ AI 직접 지휘   │  │ 기획서 정리 지원  │  │
│  └─────────────────┘  └──────────────────┘  │
│          ↕ 충돌·융합           │
│        AI (Executor)          │
│     지시받은 대로 코드를 작성하는 실행자      │
└─────────────────────────────────────────────────┘
```
- **자녀**: 두려움 없는 상상력으로 아이디어를 던지고, AI에게 직접 지시
- **부모**: 아이의 날것 같은 아이디어를 논리적 구조로 다듬어주는 현업 강사
- **결과**: 아이는 '생각을 정리하는 법'을, 부모는 '한계를 짓지 않는 용기'를 배움
