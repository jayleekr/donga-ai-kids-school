## 🎯 전체 학습 목표 (Learning Objectives)

### 지식 (Knowledge)
- [ ] AI가 "실행자(Executor)"이고, 인간이 "설계자(Architect)"임을 이해한다
- [ ] 맥락 설계(Context Engineering)의 중요성을 설명할 수 있다
- [ ] 기획서(PRD)의 구성 요소를 나열할 수 있다

### 기술 (Skills)
- [ ] AI에게 구체적이고 논리적인 지시를 내릴 수 있다
- [ ] 부모와 협업하여 아이디어를 구조화할 수 있다
- [ ] 자신의 창작물을 타인에게 발표할 수 있다

### 태도 (Attitude)
- [ ] "나도 할 수 있다"는 효능감을 갖는다
- [ ] AI를 두려워하지 않고 도구로 활용하려는 자세를 갖는다
- [ ] 부모를 "잔소리하는 사람"이 아닌 "협업 파트너"로 인식한다

### 10초 피드백 루프

```
[작성] → [저장] → [확인] → [수정 요청] → [반복]
   ↑__________________________|
         약 10초 사이클
```

**조교의 역할:**
1. 저장 버튼 눌렀는지 확인
2. 결과 확인하도록 안내
3. "뭐가 다르게 나왔어? 어떻게 고치고 싶어?" 질문
4. 막히면 프롬프트 카드 힌트 제공

4. 우리 팀 스토리: "이 아이디어는 ___에서 나왔어요" (30초)
