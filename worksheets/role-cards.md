## 📝 인쇄 가이드

### 권장 인쇄 설정

| 항목 | 설정 |
|------|------|
| **용지** | A5 또는 A4 (2장씩) |
| **인쇄** | 컬러 권장, 흑백 가능 |
| **라미네이팅** | 권장 (재사용 가능) |

### 인쇄 수량 (예: 20팀 기준)

| 카드 | 수량 |
|------|------|
| Visionary 카드 | 20장 |
| Integrator 카드 | 20장 |
| AI Executor 카드 | 10장 (2팀당 1장) |
| 팀 구조 다이어그램 | 10장 (2팀당 1장) |

### 배부 방법

1. Part 1 팀 구성 시간(13:30)에 배부
2. Visionary 카드 → 아이에게
3. Integrator 카드 → 부모에게
4. AI 카드 & 팀 구조 → 책상 위 비치
